Description
The LED class allows for control of Light Emitting Diodes
Parameters
Name Type Attributes Default Description
io object | number | string Pin identifier or IO Options (See INSTANTIATING)
device object <optional>
 {} An object containing device options
Name Type Attributes Default Description
sink boolean <optional>
 false True if the device is wired for sink drive
Examples

Using a pin number

import LED from "@j5e/led";
			
			(async function() {
			  const led = await new LED(12);
			  led.blink();
			})();

Using a pin identifier

import LED from "@j5e/led";
			
			(async function() {
			  const led = await new LED("A1");
			  led.blink();
			})();

Using an IO options object

import LED from "@j5e/led";
			
			(async function() {
			  const led = await new LED({
			    pin: 12,
			    pwm: true
			  });
			  led.blink();
			})();
